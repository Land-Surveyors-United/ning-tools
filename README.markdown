# Ning Tools

This package helps you download your data from Ning sites.

## Usage

Basic usage is to run `ningDownloadAll.sh` in your shell with the proper arguments:

    ./ningDownloadAll.sh NETWORKNAME USERNAME PASSWORD

This will result in a large ZIP file with all your data.
